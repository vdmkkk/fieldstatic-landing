# Fieldstatic Landing

A responsive Next.js landing page for Fieldstatic Electro Shield Tick Defense Spray.

## Local Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production Build

```bash
npm run build
npm run start
```

## Deploying To Vercel

Import this repository into Vercel and keep the default Next.js settings. No extra environment variables are required for the current static landing page.
